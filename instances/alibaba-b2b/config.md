# 走查配置 — Alibaba.com 国际站 B2B

## 基本信息
- 项目名称: Alibaba.com 国际站 B2B 全链路体验走查
- 目标站点: https://www.alibaba.com
- 走查范围: 核心采购链路（找-挑-询），不含下单支付

## 用户旅程阶段
- 找(Find): 首页导购、搜索、筛选
- 挑(Select): 商品详情页、供应商评估、商品对比
- 询(Inquire): 询盘表单、沟通工具、响应预期

## 用户画像 (5类)
- 小微零售商 — 40% — 采购标准品和轻定制品为主
- 中型品牌零售商 — 33.8% — 全品类覆盖，最完整
- 大型深耕品牌主 — 5% — 重定制OEM/ODM为主
- 服务提供商 — 12.1% — 标准品和轻定制品
- 制造商 — 9.1% — 找原材料/零部件

## 变量维度
- 品类: 标准品(Buy Now) / 轻定制(Logo/颜色) / 重定制(OEM/ODM)
- 新老: 老买家 / B类小白新买家 / B类资深但未用过国际站
- 地区: 美洲 / 欧洲(德国) / 亚太(印尼)
- 行业: 参数型(汽摩配) / 视觉型(服装)

## 评估重点
- AI 功能体验强化走查
- 新用户认知障碍 discovery 式探索
- 品类差异对找品和下单路径的影响

## 报告受众
- 业务老板: 关心商业机会点，AI功能体验
- 设计老板: 关心标准化流程，产品一致性
