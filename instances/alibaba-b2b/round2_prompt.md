# Round 2 执行 Prompt

**用途**: 将此 prompt 复制到支持浏览器操作的 AI Agent 中，执行地区 × 行业 overlay 走查。

**前置条件**: 建议已完成 Round 1，以便在 Round 2 中引用 R1 的发现进行深化。

**使用方式**:
1. 将下方提示词复制到 agent 对话框
2. 将三份文档内容分别粘贴到对应的 `[粘贴 XXX]` 占位符位置
3. 发送，等待 agent 开始执行

### 关键检查点：德语 AI 翻译测试

这是本轮最重要的测试项之一。确保 agent 执行以下操作：

1. 进入供应商聊天界面（Trade Manager 或 Messenger）
2. 在消息输入框中输入完整的德语测试消息（原文，不翻译）
3. 发送消息
4. 截图：
   - 发送前: `r2_s2_de_step09_german_message_sent.png`
   - 发送后/翻译结果: `r2_s2_de_step10_translation_result.png`
5. 记录 AI 翻译的触发方式（自动/手动）、翻译方向（DE→ZH？DE→EN？）、翻译质量

**如果 agent 未执行此测试，发送以下指令**：
```
你还没有执行德语 AI 翻译测试。请在德国地区走查中进入供应商聊天界面，用以下德语原文发送消息，并截图记录翻译结果：

"Guten Tag, ich möchte 500 Tassen mit meinem Firmenlogo bestellen. Können Sie mir bitte ein Angebot machen? Ich benötige auch Informationen über CE-Zertifizierung."

不要翻译这段话，直接粘贴发送。
```

### 关键检查点：375px 移动端视口

确保 agent 在印尼地区走查开始前调整视口。

**验证方式**: 要求 agent 截图时包含浏览器 DevTools 视口设置界面，或在截图中体现 375px 布局（内容明显比桌面端窄，有手机式布局）。

**如果 agent 在 1440px 下进行印尼走查，发送以下指令**：
```
印尼走查必须使用 375px 移动端视口。请：
1. 打开浏览器开发者工具（F12 或 Cmd+Option+I）
2. 点击"切换设备工具栏"（手机图标）
3. 设置宽度为 375px
4. 截图确认视口设置（r2_s3_id_step01_375px_viewport.png）
5. 然后从步骤1重新开始印尼地区走查
```
