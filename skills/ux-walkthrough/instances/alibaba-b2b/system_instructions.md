# System Instructions -- Alibaba B2B Walkthrough

## Target
- URL: https://www.alibaba.com
- Product: B2B wholesale marketplace connecting global buyers with Chinese manufacturers/suppliers

## Journey
Find → Select → Inquire

### Sub-stages
- **Find**: F1 Homepage Landing / F2 Search / F3 Results / F4 Filters
- **Select**: S1 Product Detail / S2 Supplier Profile
- **Inquire**: I1 Contact Supplier / RFQ

## Persona
- **Xiao Li**: First-time foreign trade buyer, small design studio founder. Zero Alibaba experience. Searching for customizable mugs with company logo. Discovery Mode = YES.

## Evaluation Focus
1. **Terminology clarity**: Flag all jargon (MOQ, FOB, OEM, ODM, Trade Assurance, Gold Supplier, RFQ, etc.) — would a newbie understand?
2. **Trust signals**: Can Xiao Li tell if a supplier is reliable? Are verification badges explained?
3. **Customization UX**: How easy is it to understand what's customizable, minimum quantities, and logo printing options?
4. **Price transparency**: Are prices clear? Hidden costs? Shipping estimates?
5. **Inquiry friction**: How many steps to contact a supplier or request a quote?
6. **AI features**: Evaluate any AI-powered features (smart search, recommendations, chat assistant)
7. **Information architecture**: Can a new user navigate from homepage to inquiry without getting lost?

## Platform Terminology to Flag
MOQ, FOB, CIF, EXW, OEM, ODM, Trade Assurance, Gold Supplier, Verified Supplier, RFQ, Alibaba.com Logistics, Buyer Protection, Sample Order, Lead Time, Production Capacity, Response Rate, Transaction Level

## Severity Definitions
- **P0 Blocker**: Cannot proceed with task at all (broken flow, error, login wall blocking core function)
- **P1 Severe**: Major confusion or friction that would cause most new users to abandon or make wrong decisions
- **P2 Moderate**: Noticeable friction or confusion, but user can work around it with effort
- **P3 Minor**: Small polish issues, slight confusion, cosmetic problems
